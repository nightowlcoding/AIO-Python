# Instagram Image Downloader

This project is an application that allows users to download high-quality images from Instagram. It provides a simple interface for users to input image URLs and retrieve the images directly to their local storage.

## Project Structure

```
instagram-image-downloader
├── src
│   ├── app.py
│   ├── downloader.py
│   └── utils
│       └── helpers.py
├── requirements.txt
└── README.md
```

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/instagram-image-downloader.git
   ```
   
2. Navigate to the project directory:
   ```
   cd instagram-image-downloader
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the application:
   ```
   python src/app.py
   ```

2. Open your web browser and go to `http://localhost:5000`.

3. Enter the Instagram image URL you wish to download and click the download button.
